Privacy Policy

Last updated: April 17, 2026

Overview

AirTok is a local network file transfer app. We take your privacy seriously. This app does not collect, store, or transmit any personal data to our servers.

Data Transfer

All file transfers occur exclusively within your local network. Data is transferred directly between devices in a peer-to-peer manner, without passing through any cloud server or third-party relay.

Data Collection

AirTok does not collect any of the following:

• Personal information (name, email, phone number, etc.)
• Device identifiers (IDFA, IDFV, etc.)
• Location data
• Usage analytics
• Contents of transferred files

Network Communication

AirTok uses the following network protocols, restricted to your local network only:

• UDP broadcast/multicast: to discover nearby devices on the LAN
• TCP connections: for file transfer between devices, with TLS encryption support

These communications do not involve internet access and no data is sent outside your local network. The only exception is the remote configuration check on the About page (for version updates), which fetches a public JSON file and does not send any user data.

Third-Party Services

AirTok does not integrate any third-party analytics, advertising, or tracking SDKs.

Data Storage

Received files are saved to your device's local storage (in the download directory you specify in Settings). AirTok does not upload these files anywhere.

Children's Privacy

AirTok is not directed at children under 13 and does not knowingly collect personal information from children.

Changes to This Policy

If this policy is updated, changes will be posted on this page. We encourage you to review it periodically.
